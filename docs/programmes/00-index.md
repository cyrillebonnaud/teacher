# Enseigner au cycle 4 — index


Le cycle 4 (cycle des approfondissements) recouvre les classes de 5e, 4e et 3e. Cette rubrique présente les programmes, les attendus de fin d'année et les repères annuels de progression ainsi que les ressources d'accompagnement associées.
