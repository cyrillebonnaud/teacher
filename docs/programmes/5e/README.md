## Notes

- **Disciplines avec thèmes explicitement 5e** : Histoire, Géographie, Français (culture littéraire), EMC, EVAREVARS
- **Disciplines organisées par cycle** : les contenus de Maths, Physique-Chimie, SVT, Technologie sont filtrés à partir des "repères de progressivité" qui précisent ce qui est attendu "dès la 5e" ou "en 5e"
- **Français** : un nouveau programme entre en vigueur en 5e à la rentrée 2026-2027 (BO 5 mars 2026). Le fichier `francais.md` décrit le programme actuel (applicable jusqu'en 2025-2026).
